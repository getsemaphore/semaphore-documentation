## Welcome to Semaphore

Semaphore provides a powerful API for accessing comprehensive company data across multiple jurisdictions. Our platform enables you to search, retrieve, and analyze company information efficiently and cost-effectively.

## Getting Started

Here's what you need to know to start using Semaphore's API:

<Steps>
  <Step title="Authentication">Signup at [app.getsemaphore.com](https://app.getsemaphore.com) to obtain your API key.</Step>
  <Step title="Credits">Purchase credits to access data points and documents - all requests are billed using our credit system.</Step>
  <Step title="Make Requests">Use our RESTful API endpoints to search and retrieve company data.</Step>
</Steps>
